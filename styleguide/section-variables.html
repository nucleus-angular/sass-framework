Variables

There are a number of pre-defined global variables that you can use.

Colors

These are the only color that should be used within the application.

  • $color-maroon  
  • $color-medium-carmine

Font

Variables that deal with fonts.

  • $font-size 1.3rem
  • $font-family sans-serif

Forms

These are variables used for from styles

  • $border-input 2px solid shade($color-gray, $percentage-darkest)
  • $box-shadow-input inset 0px 0px 15px $color-gallery
  • $box-shadow-focus-input inset 0px 0px 15px shade($color-gallery, $percentage-darkest)

Invalid Colors

These are colors used to indicator something is invalid.

  • $border-color-invalid  
  • $background-color-invalid  
  • $color-invalid  

Percentage

General percentage values.

  • $percentage-dark 10%
  • $percentage-darker 20%
  • $percentage-darkest 30%
  • $percentage-light 10%
  • $percentage-lighter 20%
  • $percentage-lightest 30%

Valid Colors

These are colors used to indicator something is valid.

  • $border-color-valid  
  • $background-color-valid  
  • $color-valid
