Expander

Expander is an element that toggle display of the content. The container must have the expander class and it's dom must have an element with the handle class and content class. The content is only visable when the container has a class of is-active.

Handle
Content

<div class="expander">
  <div class="handle">Handle</div>
  <div class="content">Content</div>
</div>

Handle
Content

<div class="expander is-active">
  <div class="handle">Handle</div>
  <div class="content">Content</div>
</div>
